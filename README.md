# Boarded Up
Try to survive as many rounds as possible while `Chicken Zombies` are trying to kill you! What helps you focus? The amazing yeat instrumentals in the background.

- [Gallery](#gallery)
- [Installation and setup](#installation-and-setup)
- [Future plans](#future-plans)
- [History](#history)

---

## History
I made this during my second semester at Wright State University. It was originally a class project, and later we had a game jam where we got more time to work on it. I ended up winning third place.
